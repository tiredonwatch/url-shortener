# Go URL Shortener

Hey! This is a personal project that I'm building to learn Go language and web development concepts.

As I'm actively learning while building this project, the comments are a bit descriptive about what is happening rather than why its happening.

I got the idea to make this project after I worked on the TCP Server in C.

I plan on rebuilding the C TCP Server project soon using standard programming practices and will be pushing it on Github! Its an empty repository as of now.

## Project Structure
Here's the project Structure as of now:

```
+ url-shortener
  + cmd
    + server
      + main.go
    + .gitkeep 
  + internal
    + handler
      + shorten.go
  + .gitignore
  + go.mod
  + README.md
```
